# 音频可视化工具 OpenGL 版 - 创作说明

M2 保留了原版 M 的控制逻辑、预设系统、随机系统、分区预设、Windows 系统音频捕获和五层轮廓/四层条形结构，但把底层渲染从 pygame 全面切换到了 PyOpenGL。

这次重制的重点不是只把绘图 API 换一层，而是保留原来的参数生态和使用方式，让旧预设可以继续工作，同时把渲染路径改成更适合叠加、透明、线条与填充混合输出的 OpenGL 管线。现在控制台仍然负责参数编辑与进程管理，子窗口则专门负责音频分析、曲线构建和 OpenGL 绘制。

当前版本保留了这些核心能力：

- Windows WASAPI 回采系统音频
- K1 / K2 响度驱动
- 五层轮廓与四组层间条形
- B 样条闭合轮廓
- 颜色方案、渐变、高度/频率映射、动态色相循环
- 预设保存、加载、重命名、删除、自动切换
- 分区预设与随机化系统
- 透明顶层悬浮窗口与拖动调整中心点

M2 的目标不是做一份简化版，而是让原项目现有工作流基本不变，只把渲染核心替换为更适合继续扩展的 OpenGL 方案。
